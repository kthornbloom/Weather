# Description

This is a weather app utilizing https://open-meteo.com/

## Operation
The weather app should be an extremely minimal and simple app that all focuses on a main graph. This graph charts temp and precipitation at first, but we'll have a dropdown selector to pick anything else available from the open meteo API. This graph must be REALLY REALLY GOOD, animating on entry, showing specific values if touched or clicked on a certain point, showing a line for "right now", etc.

## Theme

Use an OLED black theme (super dark mode). Use 100% white for graphs, text, etc, and use bright colors like cyan, #ebde34,
#f20232, etc for alternate graph lines or wherever color is needed. Use repeating dot patterns for texture (like pixels). Think: teenage engineering or the "nothing" phone company when looking at style. This must be brutalism, high end, minimal fashion style. No emojis, no rounded corners, no "cards".
